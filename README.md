## Description

A simple extension to ajust Adwaita and maybe other Themes Corners radius to make it more squared or round.

### Via extensions.gnome.org (Recommended)

.....

### Manual

1. Clone repo: `git clone https://github.com/dudumaroja/cornercraft.git`
2. Copy to extensions: `cp -r cornercraft@dudumaroja ~/.local/share/gnome-shell/extensions/`
3. Compile schemas: `glib-compile-schemas ~/.local/share/gnome-shell/extensions/cornercraft@dudumaroja/schemas/`
4. Restart Shell: Alt+F2, `r` (X11) or logout/login (Wayland).
5. Enable: `gnome-extensions enable cornercraft@dudumaroja`
